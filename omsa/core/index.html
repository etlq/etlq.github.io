My review of everything

[GT OMSA] All core courses should be made optional


I understand they want the program curriculum to have structure. But all 5 core courses should be made optional in my opinion. Just like OMSCS, they should allow more flexibility in the course selection.

Basic core courses

They do allow opt-out for the basic core courses (6501 iAM, 8803 BFA, 6040 iCDA). But not every one of them is easy to opt out. Especially for opting out MGT 8803, you must have previously completed university courses that cover the same content. But the 8803 content is so disconnected from the rest of the program and nothing builds on it. I think we should make it optional.

I know, I know,, the argument is analytics students should have exposure to the business concepts. But then it can be perfectly argued that, for example, analytics students should have exposure to Bayesian statistics as it provides a powerful framework for updating beliefs based on new evidence, which is particularly useful when dealing with uncertainty and prior knowledge. And therefore ISYE 6420 (Bayesian Statistics) should be mandatory. Or this argument can be made for just about any other courses in the program.

Advanced core courses

As for the mandatory advanced core courses: MGT 6203 (DAB) and CSE 6242 (DVA), having taken both (and earned As), I feel they really could be made optional too. There is nothing in these two courses that other courses depend on.

There should be more flexibility in the curriculum

If the concern is, without core courses, students may not build solid foundation to succeed in the rest of the program, well then enforce more explicit pre-requisites for each course. For example, it might look something like "To register for isye 6740, students must have successfully completed isye 6501 or show equivalent academic (or industry) experience."

The current structure of keeping 5 core courses mandatory and yet allowing students to to take whichever hard courses like HDDA, CDA, DL, DO as the first class is not making a lot of sense.

The program demographic is diverse. We should facilitate more flexible curriculum for students to customize course selection to maximize their educational benefits. To this end, all OMSA courses should be made optional, and constrained only by track requirements.

Similarly, the practicum should be made optional in my opinion.
